Clone the project

```bash
  git clone https://github.com/BenJZak/RandomHorrorStory.git
```

Install the following dependencies:

For access to Reddit's API:
```bash
$ pip install praw
```
For python based UI:
```bash
$ pip install tkinter
```
For access to Unsplash API:
```bash
$ pip install requests
```
For image handling and modification:
```bash
$ pip install pillow
```
